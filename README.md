# Psicontrol Front-end

O Psicontrol é um SaaS focado na gestão clínica de psicólogos, otimizando o agendamento, cobranças e comunicação com pacientes.

## Tecnologias
- React.js
- TypeScript
- Tailwind CSS
- Axios
- React Query

## Como rodar localmente
1. Clone o repositório: `git clone <url>`
2. Instale as dependências: `npm install`
3. Configure as variáveis de ambiente (.env)
4. Inicie o servidor: `npm run dev`

## Features
- Dashboard de gestão de consultas
- Integração com Google Agenda
- Interface responsiva para desktop e mobile
